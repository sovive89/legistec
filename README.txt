
Legistec — front-end versão inicial
Arquivos:
- index.html
- /pages/login.html
- /pages/dashboard.html
- /pages/reset.html
- /pages/error.html
- /css/style.css
- /js/firebase-init.js  (COLE AQUI O firebaseConfig DO SEU APP)
- /js/auth.js

Instruções rápidas (sem CLI):
1. Descompacte este pacote.
2. Abra o Firebase Console > seu projeto > Hosting > Upload files (ou publicar) e envie o conteúdo da pasta.
3. No Console, copie o firebaseConfig do App Web e cole em /js/firebase-init.js.
4. Ative Authentication > Sign-in method > Email/Password.
5. Crie um usuário via Authentication > Users (Adicionar usuário) ou use Sign-up via app se implementar.
6. Acesse o site hospedado e teste login e dashboard.
